# Indian Atlantic Hotel Management Platform

A premium, luxury-focused hotel management SaaS designed for the **Indian Atlantic Hotel** in Agbor. This platform features a high-end guest booking engine and a comprehensive administrative dashboard for real-time inventory and booking management.

## ✨ Features

### Guest Experience
- **Premium UI/UX:** Built with a luxury navy-and-gold aesthetic using glassmorphism.
- **Smart Booking Engine:** Real-time availability checking with anti-double-booking logic.
- **Advanced Filters:** Search rooms by price range, category, and specific stay dates.
- **Floating WhatsApp Integration:** Direct link for instant guest support.
- **Interactive Location Map:** Easy navigation to the hotel in Agbor.
- **Fully Responsive:** Optimized for mobile, tablet, and desktop.

### Admin Dashboard
- **Live Inventory Management:** Update room prices and availability status (Available, Maintenance, Booked).
- **Booking Management:** Searchable guest ledger with life-cycle tracking (Confirmed, Checked-in, Checked-out, Cancelled).
- **Payment Logs:** Automated tracking of all successful transactions.
- **KPI Overview:** Real-time metrics for total revenue, occupancy, and pending tasks.

## 🚀 Tech Stack
- **Frontend:** React.js, Vite
- **Styling:** Vanilla CSS, Framer Motion (Animations)
- **Icons:** Lucide React
- **State Management:** React Context API

## 🛠️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/indian-atlantic-hotel.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run locally:**
   ```bash
   npm run dev
   ```
